# 介绍

SoTap 作为游戏兴趣圈，必然无法缺少「论坛」这一社群体系的支持。我们开放论坛一方面是为了加强服务器内玩家的相互讨论、交流与沟通，一方面是让外来访客感受到服务器内的独特氛围。为此，我们正一步步让论坛变得更好。

为了给玩家营造一个更纯粹的讨论环境，我们并没有采用当前流行的 Discuz! 或者 XenForo，而是采用了 [Flarum](//flarum.org)。
